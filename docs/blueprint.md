# **App Name**: Handyman Hero

## Core Features:

- Service Listings: Display a comprehensive list of available handyman services, categorized for easy browsing.
- Online Booking: Allow customers to book services directly through the website with a user-friendly scheduling system.
- Before & After Gallery: Showcase previous projects with a visually appealing before-and-after gallery.
- Customer Testimonials: Display positive customer reviews to build trust and credibility.
- AI-Powered Task Analyzer: Allow users to describe a task, and use AI as a tool to suggest the most appropriate service or set of services offered.

## Style Guidelines:

- Primary color: Dark navy (#1E293B) for a trustworthy and professional feel.
- Background color: Warm white (#F8FAFC) to provide strong contrast and a clean look.
- Accent color: Teal (#14B8A6) used sparingly to highlight calls to action and important information.
- Font: 'Inter', a clean sans-serif, for both headlines and body text, ensuring readability and a modern aesthetic.
- Use simple, modern icons to represent each service, enhancing visual clarity and user experience.
- Emphasize generous whitespace and clean content sections for an uncluttered and professional design.
- Incorporate subtle hover effects on buttons and cards to add interactivity without distracting from the overall design.
